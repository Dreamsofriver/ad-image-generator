# 固定元素库

这里存放长期使用的固定元素，用于广告图片生成。

## 目录结构

```
elements/
├── icons/          # 图标文件
├── borders/        # 边框和装饰
├── backgrounds/    # 背景图案
├── buttons/        # 按钮样式
└── overlays/       # 叠加效果
```

## 元素分类

### 1. 图标 (icons/)
- **箭头图标**: 用于指示方向或行动
- **社交图标**: 社交媒体平台Logo
- **功能图标**: 电话、邮件、位置等
- **装饰图标**: 星星、心形、徽章等

### 2. 边框和装饰 (borders/)
- **简约边框**: 细线边框，适合现代风格
- **装饰边框**: 花纹边框，适合传统风格
- **角标装饰**: 角落装饰元素
- **分隔线**: 内容分隔线

### 3. 背景图案 (backgrounds/)
- **渐变背景**: 色彩渐变背景
- **纹理背景**: 纸张、布料等纹理
- **图案背景**: 几何图案、波点等
- **图片背景**: 通用背景图片

### 4. 按钮样式 (buttons/)
- **主要按钮**: 核心行动号召按钮
- **次要按钮**: 辅助操作按钮
- **文本按钮**: 只有文字的按钮
- **图标按钮**: 带图标的按钮

### 5. 叠加效果 (overlays/)
- **颜色叠加**: 半透明颜色层
- **渐变叠加**: 渐变透明层
- **模糊叠加**: 背景模糊效果
- **光影效果**: 阴影和高光

## 使用规范

### 尺寸要求
- **图标**: 24px, 32px, 48px, 64px (正方形)
- **边框**: 根据广告尺寸等比缩放
- **背景**: 1080×1920 或可平铺图案
- **按钮**: 最小240×48px，建议300×60px

### 文件格式
- **矢量图**: SVG (首选，可无损缩放)
- **位图**: PNG-24 (支持透明背景)
- **避免使用**: JPG (不支持透明)

### 命名规范
```
[类型]_[名称]_[尺寸]_[颜色].扩展名
示例: icon_arrow_right_32_blue.svg
       border_simple_1px_gray.png
       button_primary_300x60_green.svg
```

## 添加新元素

1. **设计元素**：确保符合广告规范
2. **优化文件**：压缩文件大小，移除冗余数据
3. **正确命名**：按照命名规范命名文件
4. **分类存放**：放入正确的子目录
5. **更新文档**：如有需要，更新此说明文件
