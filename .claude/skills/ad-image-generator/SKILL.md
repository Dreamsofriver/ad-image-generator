# 广告图片生成器 Skill (2026.4.15 优化版)

## 概述

本Skill用于生成金融类广告图片，基于参考图片使用Google Gemini Vision API生成相似的广告图片。经过优化，移除了复杂的预布局生成策略，采用直接上传参考图片+简单提示词的方式，提高生成成功率和效率。

## 核心功能

1. **从参考图片生成**：上传参考图片，生成风格相似的广告图片
2. **多参考图片支持**：支持同时上传多张参考图片，用于控制不同元素（如吉祥物形象）
3. **精确提示词控制**：使用详细的中文提示词指导AI生成，可精确控制文字内容和元素形象
4. **多模型支持**：自动尝试多个Gemini Vision模型，提高成功率
5. **批量生成**：支持批量处理多个参考图片
6. **生成记录**：自动保存生成信息和元数据

## 使用方法

### 1. 单张图片生成

```bash
# 使用默认参考图片（浅色.png）和默认提示词
python3 generate_ad_from_reference.py

# 指定参考图片和提示词
python3 generate_ad_from_reference.py \
  --reference "./.claude/skills/ad-image-generator/assets/example/浅色" \
  --prompt "请参考提供张参考图,生成一张相似的广告图片。必须严格限制图片尺寸为1344*768。"

# 指定模型
python3 generate_ad_from_reference.py --model "models/gemini-2.5-flash-image"

# 使用多张参考图片（控制吉祥物形象）
python3 generate_ad_from_reference.py \
  --reference "./.claude/skills/ad-image-generator/assets/example/浅色" \
  --additional "./.claude/skills/ad-image-generator/assets/elements/吉祥物姿势包/开心.png" \
  --prompt "请参考第一张图片（浅色参考图）的广告设计风格和布局，生成一张金融类广告图片。吉祥物的形象必须完全符合第二张图片（吉祥物姿势包中的图片）中的吉祥物形象。广告图片中必须包含标题大字：'AI涨乐，您的投资好搭子'，这个标题必须准确无误，一个字都不能错。整体风格要专业、现代、吸引人，适合金融投资类广告。"
```

### 2. 批量生成

```bash
# 批量生成多张图片
python3 generate_ad_from_reference.py \
  --batch \
  "./.claude/skills/ad-image-generator/assets/example/浅色" \
  "./.claude/skills/ad-image-generator/assets/example/深色"
```

## 优化经验总结

### 2026.4.15 优化要点

1. **简化生成策略**：
   - 移除了复杂的预布局生成和元素融合
   - 采用直接上传参考图片的方式

2. **模型选择优化**：
   - 发现Google Imagen模型在当前API版本中不支持`generateContent`方法
   - 改用Gemini Vision模型，成功率高
   - 实现多模型自动尝试机制

3. **错误处理改进**：
   - 添加详细的错误信息和调试输出
   - 实现模型失败后的自动重试
   - 保存完整的生成日志

4. **代码结构优化**：
   - 删除未使用的复杂脚本和依赖
   - 创建单一入口的简化生成器
   - 统一的配置管理和错误处理
   - **规范化命名**：生成的图片统一使用分类名（如深色/浅色）命名，而非随机文件名，便于分类管理。

### 标准化生成流程

#### 深色图片标准化生成流程
**适用场景**：生成深色背景的金融广告图片

**核心要求**：
1. **图片尺寸**：必须严格限制为 1344*768 像素（已在 API 配置中通过 aspect_ratio="16:9" 强制锁定，所有 prompt 中仍需保留此约束以增强效果）
2. **元素控制**：
   - 左上角logo必须和深色参考图保持一致
   - 中间标题大字必须和深色参考图完全相同
   - 吉祥物形象必须完全符合指定的吉祥物图片
   - 右下角风险提示必须直接从深色参考图照搬
3. **元素替换**：
   - 电脑屏幕、桌面等元素要替换为更有创意、更多样化的元素
   - 采用背景自然/人文风景 + 小的金融元素（如小钻石、小宝箱等）
4. **风格要求**：专业、现代、吸引人，适合金融投资类广告

**标准命令**：
```bash
python3 generate_ad_from_reference.py \
  --reference "./.claude/skills/ad-image-generator/assets/example/深色" \
  --additional "./.claude/skills/ad-image-generator/assets/elements/吉祥物姿势包/[吉祥物图片].png" \
  --prompt "请参考第一张图片（深色参考图）的广告设计风格和布局，生成一张金融类广告图片。必须严格限制图片尺寸为1344*768。电脑屏幕、桌面等元素要替换为其他更有创意、更多样化的元素，采用背景自然/人文风景 + 小的金融元素（如小钻石、小宝箱等），除了logo、标题大字、吉祥物、风险提示外，其他元素都要多加替换。吉祥物的形象必须完全符合第二张图片（吉祥物姿势包中的图片）中的吉祥物形象。广告图片中必须包含和深色参考图完全相同的中间标题大字，这个标题必须准确无误，一个字都不能错。左上角的logo必须和深色参考图的logo保持一致。右下角的'投资有风险，入市需谨慎'必须直接从深色参考图照搬，不要有任何变化。整体风格要专业、现代、吸引人，适合金融投资类广告。"

2. **元素控制**：
   - 不再需要吉祥物或任何人物元素。
   - 左上角logo必须和参考图保持一致。
   - 中间标题大字必须和参考图完全相同。

## 故障排除

### 常见问题

1. **API密钥无效**：
   ```
   ❌ Google API客户端创建失败
   ```
   解决方案：检查API密钥是否正确，是否有足够配额

   ```

   ```

   ```
