# 广告图片生成器 Skill (2026.4.15 优化版)

## 概述

本Skill用于生成金融类广告图片，基于参考图片使用Google Gemini Vision API生成相似的广告图片。经过优化，移除了复杂的预布局生成策略，采用直接上传参考图片+简单提示词的方式，提高生成成功率和效率。

## 核心功能

1. **从参考图片生成**：上传参考图片，生成风格相似的广告图片
2. **多参考图片支持**：支持同时上传多张参考图片，用于控制不同元素（如吉祥物形象）
3. **精确提示词控制**：使用详细的中文提示词指导AI生成，可精确控制文字内容和元素形象
4. **多模型支持**：自动尝试多个Gemini Vision模型，提高成功率
5. **批量生成**：支持批量处理多个参考图片
6. **生成记录**：自动保存生成信息和元数据

## 使用方法

### 1. 单张图片生成

```bash
# 使用默认参考图片（浅色.png）和默认提示词
python3 generate_ad_from_reference.py

# 指定参考图片和提示词
python3 generate_ad_from_reference.py \
  --reference "./.claude/skills/ad-image-generator/assets/example/浅色" \
  --prompt "请参考提供张参考图,生成一张相似的广告图片。必须严格限制图片尺寸为1344*768。"

# 指定模型
python3 generate_ad_from_reference.py --model "models/gemini-2.5-flash-image"

# 使用多张参考图片（控制吉祥物形象）
python3 generate_ad_from_reference.py \
  --reference "./.claude/skills/ad-image-generator/assets/example/浅色" \
  --additional "./.claude/skills/ad-image-generator/assets/elements/吉祥物姿势包/开心.png" \
  --prompt "请参考第一张图片（浅色参考图）的广告设计风格和布局，生成一张金融类广告图片。吉祥物的形象必须完全符合第二张图片（吉祥物姿势包中的图片）中的吉祥物形象。广告图片中必须包含标题大字：'AI涨乐，您的投资好搭子'，这个标题必须准确无误，一个字都不能错。整体风格要专业、现代、吸引人，适合金融投资类广告。"
```

### 2. 批量生成

```bash
# 批量生成多张图片
python3 generate_ad_from_reference.py \
```

## 优化经验总结

### 2026.4.15 优化要点

1. **简化生成策略**：
   - 移除了复杂的预布局生成和元素融合
   - 采用直接上传参考图片的方式
   - 使用简单中文提示词指导AI生成

2. **模型选择优化**：
   - 发现Google Imagen模型在当前API版本中不支持`generateContent`方法
   - 改用Gemini Vision模型，成功率高
   - 实现多模型自动尝试机制

3. **错误处理改进**：
   - 添加详细的错误信息和调试输出
   - 实现模型失败后的自动重试
   - 保存完整的生成日志

4. **代码结构优化**：
   - 删除未使用的复杂脚本和依赖
   - 创建单一入口的简化生成器
   - 统一的配置管理和错误处理
   - **规范化命名**：生成的图片统一使用分类名（如深色/浅色）命名，而非随机文件名，便于分类管理。

### 2026.4.15 新增功能要点

1. **多参考图片支持**：
   - 扩展`generate_from_reference`方法支持`additional_images`参数
   - 支持同时上传多张参考图片，用于控制不同设计元素
   - 新增`--additional`命令行参数，可指定多个额外图片路径

2. **精确元素控制**：
   - 使用吉祥物姿势包中的图片精确控制吉祥物形象
   - 在prompt中明确指定参考图片的顺序和作用
   - 实现品牌元素的一致性控制

3. **文字内容精确控制**：
   - 在prompt中明确要求特定文字内容，如"AI涨乐，您的投资好搭子"
   - 强调文字准确性，确保生成图片中的文字无误
   - 支持金融广告的特定文案要求

4. **复杂prompt编写**：
   - 编写包含多个约束条件的详细prompt

#### 高质感图片标准化生成流程
**适用场景**：生成强调高端、专业、视觉冲击力强且无需人物/吉祥物的金融广告图片

**参考图选择**：
- 参考图从 `assets/example/高质感/` 文件夹中**随机任选一张**（脚本会自动从文件夹随机选取）。
- 当前已补充多张范例（共 10 张：高质感1.png、高质感2.png，以及 05061~05092 系列 jpeg），确保每次生成都有不同的风格参考。
- 传入 `--reference` 时只需指定文件夹路径，无需指定具体文件。

**核心要求**：
1. **图片尺寸**：必须严格限制为 1344*768 像素（横版 16:9，需通过 `--aspect-ratio “16:9”` 参数强制锁定，脚本默认值为 `9:16` 竖版，不传参会导致尺寸错误）。
2. **元素控制**：
   - 不再需要吉祥物或任何人物元素。
   - 左上角logo必须和参考图保持一致。
   - 中间标题大字必须和参考图完全相同。
   - 右下角风险提示必须直接从参考图照搬。
3. **背景重构要求 - 必须彻底差异化**：
   - **绝对不要照搬参考图中的背景物体形状和颜色**。

## 注意事项

1. **API密钥**：需要有效的Google AI API密钥，且有足够的配额
2. **图片大小**：参考图片不宜过大，建议不超过5MB
3. **网络连接**：生成过程需要稳定的网络连接
4. **输出目录**：每次运行不会覆盖之前的生成结果
5. **模型可用性**：不同地区的模型可用性可能不同

## 故障排除

### 常见问题

1. **API密钥无效**：
   ```
   ❌ Google API客户端创建失败
   ```
   解决方案：检查API密钥是否正确，是否有足够配额

2. **模型不可用**：
   ```
   ❌ 模型 models/gemini-2.5-flash-image 失败: 404 NOT_FOUND
   ```
   解决方案：尝试其他模型，或检查API版本

3. **图片加载失败**：
   ```
   ❌ 参考图片加载失败
   ```
   解决方案：检查图片路径和文件权限

4. **无图片数据返回**：
   ```
   ⚠️ 模型响应中没有图片数据
   ```
   解决方案：调整提示词，或尝试其他模型
